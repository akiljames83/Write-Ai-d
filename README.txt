Hello all,

This program was created to address the needs of a client who has a condition called Primary General Dystonia.
The machine learning algorithm was developed in 3 weeks with minimal training. Using this program, the hand-writing of 
a client can be analyzed.

The code to make letters is the the create letters folder. The code to run the MLA is in the Code to run WA folder.

Additionally, a pdf for the dictionary of the writing for the main letters the algorithm analyzes is given.
